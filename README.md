# Software monitor

Esse é um software simples para monitoramento de servidores linux via interface web, focado principalmente no monitoramento dos servidores pela rede, como a interface é web, isso possibilita o monitoramento remoto, pode ser feito via tunel Cloudflare ou com um IP fixo.

## :clipboard: Features
- Monitoramento remoto
- Atualização em tempo real dos valores de rede, memória, CPU e discos.
